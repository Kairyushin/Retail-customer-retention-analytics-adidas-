# Retail-Customer-Retention-Analytics-Adidas-
## Project Overview
This project analyzes customer retention and repeat purchase behavior using an Adidas retail dataset.
The analysis was performed using Power BI to identify retention trends and customer behavior patterns.

## Tools Used
- Power BI
- Power Query
- DAX

## Key Insights
- Identified retention patterns across customer segments
- Analyzed repeat purchase behavior
- Built KPIs to track customer retention metrics
